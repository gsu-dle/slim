# gsu-dle/slim

Slim 4 components, commonly used third-party web libraries, and GSU extensions.

## Install:

Via [Composer](https://getcomposer.org)

```bash
$ composer require gsu-dle/slim
```

Requires PHP 8.1 or newer.
